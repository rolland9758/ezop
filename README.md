Ezop is a Windows app for people who want a simple way to manage, view, and follow stories made by different agents. It gives you a clear place to open the app, work with story content, and keep each agent’s work easy to track.

Use ezop if you want:
- One place to view agent stories
- A simple Windows app with a clean layout
- A fast way to open and use your files
- A tool that helps you keep story work in order

## ❓ Help with common issues

### The app will not open
Try these steps:
- Make sure the download finished
- Extract the files if they are in a zip archive
- Right-click the app and try Run as administrator
- Check that your Windows version is up to date

### Windows blocks the file
If Windows shows a security prompt, open the file from the release page again and make sure you downloaded the latest release file.

### The window looks too small
Use the window edge to resize it, or check your Windows display scale settings.

### Your files do not appear
Make sure you selected the right folder and that your story files are in a format ezop can read.
